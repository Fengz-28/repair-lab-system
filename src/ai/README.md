# AI

La IA local se integrara por contrato, inicialmente con Ollama. No debe tomar decisiones criticas automaticamente.

Casos futuros:

- resumen de tickets,
- respuestas sugeridas,
- busqueda semantica,
- knowledge base,
- generacion de posts,
- asistencia tecnica.

Requisitos:

- registrar `AIInteraction`,
- no enviar secretos al modelo,
- revisar sugerencias por staff,
- usar pgvector solo cuando exista corpus suficiente.
